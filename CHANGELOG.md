# Changelog

All notable changes to ZeroTV.

## [1.0.0] - 2026-02-09

### Added
- M3U/M3U8 playlist support
- HTML5 video player with HLS.js
- EPG program guide
- Auto-reconnect
- Stream health monitoring
- Custom headers
- Favorites & search
- Keyboard shortcuts
- Settings export/import

### Performance
- Fast Rust M3U parser
- Optimized RAM usage
- Small app size (~10 MB)

### Platform
- Windows 10/11 support
